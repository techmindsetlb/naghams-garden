# 🌻 Nagham's Productivity Garden

The ultimate task manager designed for **Nagham** — a multi-talented creative entrepreneur managing:
- 🎬 **WithNagham** — Content creation & Instagram
- 🕯️ **Ehsas Store** — Candles & lotions
- ☕ **Oreo Cafe** — Her brother's cafe
- 💻 **Tech Mindset LB** — Tech content & software solutions
- 📋 **More Clients** — Social media management

## ✨ Features

- **📝 Kanban Boards** — Separate boards for each project/client
- **🎨 Beautiful Design** — Pink & yellow sunflower theme with smooth animations
- **↔️ Drag & Drop** — Move tasks between boards (desktop + mobile touch)
- **🏷️ Smart Tags** — Work, Content, Store, Meeting, Urgent, Idea, Personal
- **🔴 Priority Levels** — High / Medium / Low with color indicators
- **✅ Task Completion** — Check off completed tasks with satisfaction
- **💾 Auto-Save** — All data persists in your browser's local storage
- **📦 Backup System** — Export/Import JSON backups so you never lose anything
- **📱 Fully Responsive** — Works beautifully on phone, tablet, and desktop
- **♿ Accessible** — Keyboard navigation, focus trapping, reduced motion support
- **🎬 Animations** — Staggered entries, hover lifts, smooth modals, floating sunflowers

## 🛠️ How to Use

1. Open the [live site](https://techmindsetlb.github.io/naghams-garden/)
2. Start adding tasks to the pre-made boards
3. Drag tasks between boards to organize
4. Use the **Data** button to backup or restore your data
5. Everything auto-saves — no login needed!

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl + K` | New Task |
| `Ctrl + Shift + N` | New Board |
| `Escape` | Close Modal |
| `Enter` | Submit Form |
